# Quantitative Portfolio Analytics Engine

## Overview
This repository contains an institutional-grade, Object-Oriented (OOP) framework for portfolio performance evaluation, benchmarking, and tail-risk analysis. 

The system transitions away from standard procedural scripts into a modular architecture, separating the ETL (Extract, Transform, Load) pipelines from the mathematical risk engines. It is designed to evaluate passive investment portfolios or active trading strategies against multiple benchmarks (e.g., Vanguard Total World Stock, Local Risk-Free Rates).

## Architecture & Design
The codebase is structured around the `PortfolioMetrics` class, emphasizing Separation of Concerns and Statefulness.
- **Robust Data Ingestion:** Automated alignment of time-series data handling different publishing frequencies and non-synchronous trading days.
- **Dynamic Context:** Built with `self` containment and dependency injection, allowing the mathematical engine to process static historical data or ingest dynamic equity curves generated by external Machine Learning models (e.g., XGBoost predictive signals).
- **Type Safety:** Implements Python's `typing` module to ensure stability across complex mathematical matrix operations.

## Key Quantitative Metrics
The analytical core goes beyond standard normally-distributed assumptions, focusing heavily on Tail Risk and asymmetric payoff profiles:

* **Adjusted Sortino Ratio:** Evaluated against a dynamic Minimum Acceptable Return (MAR) anchored to the Risk-Free rate, rather than absolute zero.
* **Expected Shortfall (CVaR 95%):** Quantifies the average expected loss within the worst 5% of market scenarios.
* **Win/Loss Asymmetry:** Measures the behavioral skew of the portfolio's positive vs. negative periods.
* **Rolling Risk Metrics:** Calculates 36-month Rolling Beta and Rolling Sharpe to identify regime changes and correlation breakdowns during systemic shocks.
* **Distribution Shape Analysis:** Incorporates Skewness and Kurtosis to mathematically prove the existence of fat tails in the return distribution.

## Visual Tear Sheet
The framework automatically generates professional visualization suites, including:
- **Log-Scaled Cumulative Wealth Curves** (Portfolio vs. Benchmarks)
- **Underwater Plots** to map the depth and duration of historical Drawdowns.
- **Rolling Regime Charts** (Beta & Sharpe)
- **Annual Returns Bar/Heatmaps**
